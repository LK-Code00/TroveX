# 🗂️ TroveX

A personal tool to store, search and manage scripts — built just for fun while practicing React and Supabase.

---

## 🤔 Why does this exist?

No big reason. I wanted to test my basic programming skills, understand how frontend + database integration works, and honestly... just kill some time building something real instead of following yet another tutorial.

The "scripts" part? Yeah, I use cheats in some games. Not proud, not ashamed. Just vibing. 😅

---

## 🛠️ What I learned building this

- How to structure a React project with Vite
- How to connect a frontend app to a real database (Supabase)
- The difference between a single HTML file and a proper project structure
- Why JSX exists and what Babel actually does under the hood
- How to deploy with GitHub + Vercel without touching a terminal

---

## 🚀 Tech Stack

- **React** + **Vite** — frontend
- **Supabase** — database and backend
- **Vercel** — hosting and deploy

---

## ✨ Features

- Add, search and delete scripts
- Data persists in the cloud via Supabase
- Clean dark UI with subtle animations
- Works on any device
